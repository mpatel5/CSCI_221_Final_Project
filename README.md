# CSCI 221 Final Project

## Part 1: Multi-Linear Regression (Python and R)
This section performs multi-linear regression on the developer burnout dataset using both Python and R.
The analysis includes:

* Data cleaning and preparation
* Duplicate and missing value handling
* IQR and Z-score outlier detection
* Median imputation for missing numeric values
* Multi-variable regression using five predictor variables
* Model evaluation with R², Adjusted R², and RMSE
* Coefficient interpretation and statistical significance
* R² as a percentage
* Comparison of results between Python and R

## Part 2: K-Means Clustering (Python)
This section performs K-Means clustering on the music selections dataset using Python.
The analysis includes:

* Data cleaning and preparation
* Duplicate and missing value handling
* Z-score outlier detection
* Log-transformation of skewed features
* Feature standardization with StandardScaler
* Optimal cluster selection using the Elbow Method
* K-Means clustering with K=7
* PCA-based cluster visualization
* Cluster profile interpretation

## Part 3: Association Rule Mining (R)
This section uses R to perform market basket analysis on the online store dataset.
The analysis includes:

* Data cleaning and preparation
* Duplicate removal and missing value handling
* Z-score outlier detection
* Transaction basket construction by Invoice
* Apriori association rule mining
* Support, confidence, and lift thresholds
* Network and grouped visualizations
* Interpretation of five meaningful rules
